# Prime Notes

A simple and aesthetic notes app using **localStorage**.
Features a modern UI, sidebar navigation, and easy note management.

## Features

* Add, delete, and search notes
* Tag support for notes
* Sidebar for quick navigation
* Responsive and clean UI
* Saves notes in localStorage

## Installation

1. Clone the repository:

```bash
git clone https://github.com/arhamrajputrr-commits/Prime-Notes.git
```

2. Open `index.html` in your browser.

No backend required â€” everything runs in the browser.

## Usage

1. Click **+** to add a new note.
2. Fill in **title**, **content**, and optional **tags**.
3. Use the **search bar** to filter notes by title, content, or tags.
4. Click **Delete** to remove a note.
5. Toggle between **Light** and **Dark** mode using the theme button.
